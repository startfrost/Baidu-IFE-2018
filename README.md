# Baidu-IFE-2018
这是一份2018年百度前端技术学院-基础学院任务的代码实现。任务地址：http://ife.baidu.com/course/all  
报名后进到任务地址，点击基础学院便可以看到代码对应的任务(该部分需要登录百度帐号，有时也会因为百度帐号没有头像而报名失败)。
